Project Title: Sales Data Analysis Dashboard

Project Overview : This project analyzes retail sales data to identify trends, top-performing regions, and high-value customers.The goal is to convert raw sales data into clear business insights using Power BI.

Dashboard Preview : dashboard.png

Dataset used: Sample Superstore dataset

Contains:
- Order Date
- Sales
- Region
- Customer Name
- Category

Tools Used:- Power BI
- Excel / CSV dataset

## Key Insights

- Sales are concentrated in specific regions, indicating uneven market performance
- A small number of customers contribute to a large portion of total revenue
- Sales show fluctuations over time, highlighting seasonal trends
- Certain regions consistently outperform others in revenue generation

## Business Recommendations

- Focus marketing efforts on underperforming regions
- Strengthen relationships with top customers to increase retention
- Plan inventory and campaigns based on sales trends
- Identify factors driving high-performing regions and replicate strategies

## What I Learned

- Building interactive dashboards in Power BI
- Data cleaning and transformation
- Identifying patterns and trends from raw data
- Presenting data in a business-focused way
